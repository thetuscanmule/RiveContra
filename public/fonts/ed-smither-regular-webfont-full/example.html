Getting Started

Get this font working on your site in three steps.

Step 1: Unzip and explore

Open the zip file. Inside you'll find two folders:

  • example/ — this HTML preview showing the font at different sizes
  • webfonts/ — the font files and CSS you'll add to your site

You only need the webfonts folder to go live.

Step 2: Upload the webfonts folder

Upload the entire webfonts folder to your website alongside your existing CSS files, for example:

/css/webfonts/

Keep all the files together — don't move or rename anything inside the folder.

Step 3: Add the font to your code

Paste the following inside the <head> of your HTML, after any existing stylesheets:

<link rel="stylesheet" href="/css/webfonts/ed-smither-regular.css">

That's it. Use the font anywhere in your CSS:

font-family: 'EDSmither-Regular', Helvetica, Arial, sans-serif;

Quick Troubleshooting

Font not showing up?
Double-check the webfonts folder is in the right location and the path in the CSS link matches.
Using WordPress or a site builder?
Paste the CSS into your theme's custom CSS field instead of editing HTML directly.
Web Fonts — HTML/CSS Use Only
Web fonts are licensed and built for use in websites and HTML/CSS projects only. They won't work in desktop apps like Photoshop, Illustrator, Canva, or Word. For that kind of use, you need the desktop font license — available on the same product page.
